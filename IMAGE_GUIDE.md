## Image Sourcing Options

### Option 1: Stock Photography
- **Unsplash** (free): Search for "African fashion", "Nigerian fashion", "ankara", "aso-ebi"
- **Pexels** (free): Similar searches
- **Shutterstock/Adobe Stock** (paid): More variety and quality

### Option 2: AI Image Generation
- **Midjourney**: Best for fashion photography
- **DALL-E 3**: Good for product shots
- **Stable Diffusion**: Free alternative

### Option 3: Placeholder Images
For development, you can use:
- https://picsum.photos/800/1000 (random images)
- https://placeholder.com/800x1000 (solid color placeholders)

### Option 4: Hire a Photographer
For authentic Nigerian fashion content, consider:
- Local Lagos fashion photographers
- Product photography for actual garments
- Model shoots for lookbook

## Nigerian Fashion Photography Tips

1. **Colors:** Embrace vibrant colors - emerald, burgundy, gold, royal blue
2. **Fabrics:** Showcase texture - lace detail, ankara patterns, aso-oke weaving
3. **Setting:** Lagos urban backdrop or clean studio with good lighting
4. **Models:** Diverse Nigerian beauty, confident poses
5. **Styling:** Mix traditional and contemporary elements
6. **Authenticity:** Real Nigerian cultural elements, not generic "African" imagery

## Quick Implementation

To get started immediately, I recommend:
1. Use Unsplash/Pexels for free high-quality images
2. Search terms: "African fashion model", "Nigerian dress", "ankara fabric", "luxury fashion"
3. Download and rename according to this guide
4. Place all images in `public/images/` folder

## Image Optimization

Before uploading, optimize images:
- Use WebP format for better performance
- Compress to reduce file size (aim for <200KB per image)
- Use tools like TinyPNG or Squoosh
